# Employee-Registration-System

## Employee Management System | Java, Servlets, JDBC, MySQL, HTML, CSS

A full-stack web-based Employee Management System developed using Java technologies.

## 🚀 Features
- Implemented complete CRUD operations (Create, Read, Update, Delete) for employee management
- Added employee registration with form validation and data storage
- Built dynamic employee listing page with search functionality
- Enabled update and delete operations with real-time database interaction
- Designed clean and responsive UI using HTML and CSS
- Integrated MySQL database for persistent data storage
- Deployed using Apache Tomcat server

## 🛠️ Tech Stack
- Java (Servlets, JSP concepts)
- JDBC
- MySQL
- HTML, CSS
- Apache Tomcat

## 📌 Project Highlights
- Follows MVC-style architecture
- Real-world database-driven application
- Beginner-friendly yet internship-level project
- Fully functional CRUD system with search feature
